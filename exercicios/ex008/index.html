Principais formatações

Negrito / Destaque

Nesta frase, tenho um termo em destaque usando a tag <strong> (semântica).

Nesta outra, outro termo em destaque, porém usando a tag <b> (não semântica).

Agora, utilizamos a tag <i> (não semântica) para este termo. E a tag <em> (semântica) para outro termo.

Texto Marcado

Para marcar textos utilizamos a tag <mark>. A cor pode ser formatada por individualmente como em azul, ou para todas as marcações como em rosa, rosa.

Texto grande e texto pequeno

A tag <big> que deixa o texto maior não está mais em uso. A tag <small>, ao contrário, ainda está viva na linguagem. Esta deixa o texto menor. O Prof. levanta a possibilidade de ser isso devido ao fato de que letras pequenas tem um valor semântico maior, associado às letras usadas em contratos. As letras grandes são meras opções de forma.

Texto deletado

Utilizamos a tag <del> para indicar que um texto foi desconsiderado pelo autor.

Texto inserido

Utilizamos a tag <ins> para indicar tratar-se de um texto que deve ser lido, a que se deve prestar atenção. Antigamente utilizava-se a tag <u>.

Texto sobrescrito e subscrito

Para sobrescrever um texto, usamos a tag <sup>, e para subscrever usamos a tag <sub>. Exemplo x2 e CO2.

Trechos de código

Sem a organização dos espaçamentos, apenas utilizando a tag <code>, temos a distribuição mono-espaçada das letras, que facilita e leitura de códigos: num=int(input('Digite um número'))

Se quisermos manter os espaçamentos que facilitam ainda mais a leitura do código, podemos utilizar a tag <pre>, de pré-formatado:


    num = int(input('Digite um número'))
    if num % 2 == 0:
        print(f'O número {num} é par')
    else:
        print(f'O número {num} é ímpar')
    print('FIM DO PROGRAMA')

Citações

Para citações, podemos utilizar duas tags. A tag <q> (de quote) para citações mais curtas, de meio de parágrafo. Por exemplo, a frase se beber não dirija.

Podemos também querer fazer citações mais longas, neste caso utilizamos a tag <blockquote>. No exemplo:

O PHP é uma linguagem para a criação de scripts para a Web do lado servidor embutidos em HTML, cujo código-font é aberto, e que é compatível com os mais importantes servidores Web (especialmente Apache).

O parâmetro cite não afeta nata na experiência visual da página. Porém ajuda ferramentas de busca como o google a relacionarem a citação com um endereço.
